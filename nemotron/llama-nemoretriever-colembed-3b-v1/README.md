# NVIDIA Llama NemoRetriever ColEmbed 3B V1

This is a traditional [Truss](https://truss.baseten.co/) implementation for [NVIDIA's Llama NemoRetriever ColEmbed 3B V1](https://huggingface.co/nvidia/llama-nemoretriever-colembed-3b-v1) cross-modal embedding model. This implementation uses the standard `load()` and `predict()` pattern with a `model.py` file.
